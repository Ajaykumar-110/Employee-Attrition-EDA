## Overview
This project performs Exploratory Data Analysis (EDA) on an HR dataset
to identify factors influencing employee attrition.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Key Steps
- Data cleaning and preprocessing
- Univariate and bivariate analysis
- Outlier detection
- Correlation analysis
- Data visualization
- Business insights

## Key Insights
- Attrition is higher in early career stages
- Low job satisfaction increases attrition
- Sales department shows higher attrition

## Conclusion
This project helps HR teams improve retention strategies
using data-driven insights.
